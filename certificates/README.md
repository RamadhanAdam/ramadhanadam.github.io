# Certificates

Upload certificate PDFs or images into this folder, then add them to `index.json`.

Example:

```json
[
  {
    "title": "Certificate Name",
    "issuer": "Issuer Name",
    "date": "2026",
    "file": "certificate-file.pdf",
    "description": "Optional short note."
  }
]
```

Use `url` instead of `file` when the certificate lives on another website.
